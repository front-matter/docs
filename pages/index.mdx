# Open Source Software

Front Matter has released the following open source software:

### [Briard](https://github.com/front-matter/briard)

Ruby gem and command-line utility for conversion of DOI metadata from and to different metadata formats, including schema.org.

### [Headline](https://github.com/front-matter/Headline)

Fork of the official Ghost theme, with improved display of metadata such as the canonical URL and license.
